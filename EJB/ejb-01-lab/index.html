EJB local and remote calls

This implements demo showing local EJB calls and remote EJB access from external JMV instance.
Server

This project defines various bean scopes Try to use following injection combinations and check the bean behavior

1. Inject DependentBean into
- Singleton
- ApplicationScoped
2. Inject DependentBean into
- SessionScoped
- RequestScoped
Client

This project has compilation and runtime dependency on ejb-01-lab project. It it also necessary to add ApplicationUser into wildfly realm. In order to do that, use bin/add-user.sh script. Created user ejbuser with password ejbuser123.
