XML and JSON processing

XML and JSON Processing

  • XML and JSON is used to exchange data and events between heterogeneous systems.
  • Java SE and EE has subsystems to deal with XML and JSON transformations.
    • JAX-P: Java API for XML processing
    • JAX-B: Java Architecture for XML binding
    • JSON-P: Java API for JSON processing
  • JAX-* and JSON-P are workhorses of data transformations widely used in all Java applications.
  • Used in WebServices, PrimeFaces, ...

JAX-P

  • Internally uses DOM and SAX XML parsing.
  • Supports XSLT
  • Supports DTD validations

PackageDescription

javax.xml.transformDefines the XSLT APIs that let you transform XML into other forms.

JAX-P DOM

  • Tree-like access to XML data structures.
  • Uses javax.xml.parsers.DocumentBuilderFactory class to get a DocumentBuilder instance
  • Easy to add or remove XML elements
  • DOM manipulation using Java APIs

JAX-P XSLT

  • TransformerFactory object is used to create a Transformer.
  • Source Object may be created from SAX, DOM or inputstream.
  • Result object can be SAX event handler, a DOM, or an output stream.

JAX-P StAX

  • StAX is the latest API in the JAXP family, and provides an alternative to SAX, DOM, TrAX
  • For developers looking to do high-performance stream filtering, processing, and modification,
  • Low memory and limited extensibility requirements.
  • Bidirectional pull parser interface for streaming XML processing
  • Offering a simpler programming model than SAX and more efficient memory management than DOM.

JAX-B - goals

  • Full W3C XML Schema support
  • Binding existing Java classes to generated XML schema
  • Ease of Development

JAX-P

  • JSON: JavaScript Object Notation
  • Set of Java APIs to parse, generate, transform, and query JSON.


PackageDescription
javax.jsonProvides an object model API to process JSON.
javax.json.spiService Provider Interface (SPI) to plug in implementations for JSON processing objects.
javax.json.streamProvides a streaming API to parse and generate JSON.

JSON-P Object Model API

  • API provides a random-access, tree-like structure that represents the JSON data in memory.
  • Very similar to XML DOM approach.
  • Streaming API provides a way to parse and generate JSON in a streaming fashion.
