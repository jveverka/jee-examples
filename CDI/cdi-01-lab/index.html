JEE7 / CDI / cdi-01

Simple CDI demo

This lab project demonstrates beans with different CDI scopes deployed into application server. Compile and deploy the project into AS

  1. load URL http://localhost:8080/cdi-01-lab/
  2. check the log file of the application server and look for
    3. life-cycle callbacks of different bean scopes
    @PostConstruct
    @PreDestroy
  3. load same URL in new browser tab
  4. load same URL in different instance of browser, use IE and FF or Chrome
  5. check the log file of the application server and look for life-cycle callbacks of different bean scopes
  6. statistics for created beans are displayed on URL http://localhost:8080/cdi-01-lab/admin.xhtml
  7. conversation demo is available at URL http://localhost:8080/cdi-01-lab/conversation.xhtml
