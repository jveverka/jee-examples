Protocol Upgrade demo

This is simple example implementation of HTTP 1.1 protocol upgrade. dem o consists of 3 parts.
  • protocol-upgrade-common - common utilities for data read/write into/from data streams
  • protocol-upgrade-server - sevlet implementing protocol handler on server side
  • protocol-upgrade-client - client using special protocol to communicate with server
Build and install the demo 

        cd protocol-upgrade-common
        gradle clean install
        cd protocol-upgrade-server
        gradle clean war
        cd protocol-upgrade-client
        gradle clean jar
