JAX-RS Swagger demo on WildFly

This demo uses swagger with WildFly REST endpoint. Notable URLs:
  • Main Page - http://localhost:8080/servlet-09_jaxrs-swagger/index.html
  • Swagger UI - http://localhost:8080/servlet-09_jaxrs-swagger/dist/index.html
  • REST GET info - http://localhost:8080/servlet-09_jaxrs-swagger/ws/swaggers/info
  • REST GET data - http://localhost:8080/servlet-09_jaxrs-swagger/ws/swaggers/data?data=zzzz
  • Swagger JSON - http://localhost:8080/servlet-09_jaxrs-swagger/ws/swagger.json
