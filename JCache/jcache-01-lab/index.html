JCache demo on WildFly

This is simple JCache example (Infinispan). http://localhost:8080/jcache-01-lab/ is simple UI where you can write and read jcache. Try this from different browser sessions.

this example requires Wiladflye AS infinispan subsystem configuration: 

<subsystem xmlns="urn:jboss:domain:infinispan:4.0">
    <cache-container name="jcacheTest" default-cache="jcacheTestdb">
        <local-cache name="jcacheTestdb">
            <transaction mode="NON_XA"/>
        </local-cache>
    </cache-container>
</subsystem>

ite.examples.jcache.services.CacheAccessBean stateless bean is facade interface for transactional (container-managed transaction) cache access in case 'null' or empty key is attempted to insert into cache, NPE is thrown causing transaction abort and data is not stored in cache.

note that file META-INF/MANIFEST.MF has to have dependency on org.infinispan dependencies: org.infinispan export
