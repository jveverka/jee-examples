JEE7 / Hybrid Application

This application is technical demo for:

  • real-time client-server communication using websocktes
  • integration html5, javaSE and javaFX client with wildFly 10 AS
  • google protcol buffers or JSON are used for transport via rest APIS and websockets
  • http and websocket session synchronization

Projects

  1. hybrid-app-common - communications protocols (using protobuf) and JAX-RS provider implementations
  2. hybrid-app-common-client - common base for html and websocket clients with basic integration tests
  3. hybrid-app-javafx - JavaFX client implementation, JavaSE
  4. hybrid-app-rpi - Raspberry PI client implementation, JavaSE, requires itx-rpi-drivers project
  5. hybrid-app-server - Backend implementation deployable into WildFly 10 AS

Projects dependencies

  • hybrid-app-javafx - depends on hybrid-app-common, hybrid-app-common-client
  • hybrid-app-rpi - depends on hybrid-app-common, hybrid-app-common-client, itx-rpi-drivers
  • hybrid-app-server - depends on hybrid-app-common

How to compile


gradle --project-dir hybrid-app-common clean install
gradle --project-dir hybrid-app-common-client clean install
gradle --project-dir hybrid-app-javafx clean jar
gradle --project-dir hybrid-app-rpi clean jar
gradle --project-dir hybrid-app-server clean war
