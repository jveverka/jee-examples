JSF + JS demo

This demo contains two functionally identical JSF views.
JSF + JS table data view, combination of JavaScript and JSF technology to display table data.
JSF only table data view, pure JSF view to display same table data.

  • JSF + JS table data view
  • JSF only table data view

Data is session scoped. Each session has same start data. Data is not persisted.
